# Automotive Software Engineering Practicals

1. Unit (Introduction):
To get to know the Board
2. Unit (CAN1): 
Easy CAN, Connection ECU/PC, R/W CAN
3. Unit (CAN2):
Teamwork, Bus-System CAN
4. Unit(AUTOSAR):
AUTOSAR Application
5. Unit (Test): 
Black Box Test ECU, manual an automatic
